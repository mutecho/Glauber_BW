## 实现目标

从每个事件的 participant 点云 `{(x_i, y_i, w_i)}` 构造协方差椭圆，并对任意发射点 `(x, y)` 计算默认横向流场。当前实现固定 `w_i = 1`，保持 `events.eps2` / `events.psi2` 的历史物理语义不变。

## 事件级椭圆

1. 先求带权质心：
   - `centerX = sum(w_i * x_i) / sum(w_i)`
   - `centerY = sum(w_i * y_i) / sum(w_i)`
2. 再求中心化二阶矩：
   - `sigmaX2 = sum(w_i * dx_i^2) / sum(w_i)`
   - `sigmaY2 = sum(w_i * dy_i^2) / sum(w_i)`
   - `sigmaXY = sum(w_i * dx_i * dy_i) / sum(w_i)`
3. 对 `2 x 2` 协方差矩阵做解析对角化，得到：
   - `lambdaMajor >= lambdaMinor`
   - `radiusMajor = sqrt(lambdaMajor)`
   - `radiusMinor = sqrt(lambdaMinor)`
   - 长轴 `e_major`
   - 短轴 `e_minor`
4. 继续用现有摘要定义写回：
   - `eccX = sigmaY2 - sigmaX2`
   - `eccY = 2 * sigmaXY`
   - `eps2 = sqrt(eccX^2 + eccY^2) / (sigmaX2 + sigmaY2)`
   - `psi2 = 0.5 * atan2(eccY, eccX)`
5. 椭圆有效性要求：
   - `Npart >= 2`
   - `trace > 1e-12`
   - `lambdaMajor > 1e-12`
   - `lambdaMinor > 1e-12`

无效事件仍保留质心、协方差和 `eps2/psi2`，但流场评估退化为零横向流。

## 默认流场

对每个发射点 `(x, y)`：

1. 先减去事件质心并投影到主轴系：
   - `x' = e_major · (x - centerX, y - centerY)`
   - `y' = e_minor · (x - centerX, y - centerY)`
2. 定义归一化椭圆半径：
   - `rTilde = sqrt(x'^2 / radiusMajor^2 + y'^2 / radiusMinor^2)`
3. 定义椭圆法向角：
   - `phiB = atan2(y' / radiusMinor^2, x' / radiusMajor^2)`
4. 定义横向流快度：
   - `rhoRaw = pow(rTilde, flowPower) * (rho0 + rho2 * eps2 * cos(2 * phiB))`
5. 横向流大小采用：
   - `betaT = tanh(max(0, rhoRaw))`
   - 再裁到 `< 1` 的安全上限 `0.95`
6. 横向方向沿椭圆法向，随后与 `eta_s` 的 Bjorken-like 纵向流组合。

这意味着当前默认模型已经不是“实验室原点径向流 + 标量调制”，而是“事件协方差椭圆法向流”。

## Debug 输出

仅当 `debug-flow-ellipse = true` 或 CLI 传入 `--debug-flow-ellipse` 时，才额外写出：

- `flow_ellipse_debug`
  逐事件树，记录质心、协方差、本征值、半轴、主轴、`eps2`、`psi2` 和 `valid`
- `flow_ellipse_participant_norm_x-y`
  把所有 `valid` 事件的 participant 做“质心平移 -> 主轴投影 -> 半轴归一化”后的汇总 `TH2`

当前实现不再写“每事件一张二维 debug 图”。
