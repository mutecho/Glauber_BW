## 简介
这是一个用 C++17 和 ROOT 实现的小型 Blast-Wave 事例生成器。它的目标不是做完整重离子末态模拟，而是在一个足够小、足够清晰、可继续扩展的代码骨架里，完成以下事情：

- 用 Monte Carlo Glauber 产生逐事例的 participant 几何涨落
- 在 participant hotspot 上做简化的 blast-wave 发射
- 通过 ROOT 输出事件树、粒子树和基础 QA 直方图
- 提供一个独立 QA 程序检查输出是否合法、接口是否稳定

目前末态粒子只有pi+

## 项目主要思路：

1. Glauber Model 采样核内核子分布
2. 给定impact patameter，对核做偏移，确定overlap
3. overlap内，如果核子之间的距离小于给定截面半径，则发生碰撞成为participant
4. participant 相当于受损核子，有两个分支：
   1. participant直接作为源场点
   2. participant演化成末态场，再进行下一步
5. 在做好的源中进行末态粒子生成，其中：
   1. 发射粒子数由NBD（负二项分布）确定，参数预先给定
   2. 粒子的发射位置（流体元）由核子位置+Gaussian确定（相当于发射源的分布），如果加入演化，则为演化后的核子位置为中心
   3. 粒子速度（动量）由两部分确定：
      1. 流体元速度
      2. 流体元静止系中粒子速度
      3. 后者boost到实验室系中得到粒子详细情况
