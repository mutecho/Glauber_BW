V1 方案：Glauber-density response + anisotropic blast-wave emission

实现状态说明：当前代码已按 V1a 固定参数落地，公开二阶响应输入为 `kappa2`。二阶流响应使用初态 participant eccentricity vector，即 `kappa2 * eps2_initial` 和 `psi2_initial`；`eps2_f/psi2_f/chi2` 只作为 freeze-out 几何诊断和发射源描述。

目标

构建一个轻量的 freeze-out source generator，使模型满足：

\text{initial participants}
\rightarrow
s_0(x,y)
\rightarrow
s_f(x,y)
\rightarrow
u^\mu(x,y,\eta_s)
\rightarrow
\text{particle emission}

其中：

* s_0(x,y)：初态横向熵密度或发射权重密度；
* s_f(x,y)：经过膨胀与平滑后的 freeze-out 横向发射密度；
* u^\mu：局域流体四速度；
* 粒子在局域静止系中热抽样，再 Lorentz boost 到实验室系。

核心原则：

s_f(x,y)\neq s_0(x,y)

但也不做完整 hydro / AMPT / EPOS，而是用一个可控的 response layer 模拟横向膨胀、偏心率衰减和流场响应。

⸻

1. 初态密度构建（代码中已有）

从 Woods-Saxon / NLEFT / 其他核结构模型抽样核子位置，并根据 Glauber 或已有 overlap 规则选出参与碰撞的 participants。

对每个 participant 做 Gaussian smear，得到初态横向密度：

s_0(x,y)
=
\sum_{i\in {\rm part}}
w_i
\exp\left[
-\frac{(x-x_i)^2+(y-y_i)^2}{2\sigma_{\rm dep}^2}
\right]

其中：

* x_i,y_i：第 i 个 participant 的横向位置；
* w_i：沉积权重，第一版可取 w_i=1；
* \sigma_{\rm dep}：初态沉积平滑宽度。

这一层代表初始熵密度或初始发射权重，不直接作为 freeze-out 源。

⸻

2. 初态几何谐波计算（代码中已有）

从 s_0(x,y) 计算事件级空间偏心率：

\mathcal E_n
=
\epsilon_n e^{in\Psi_n}
=
-
\frac{
\int r^n e^{in\phi}s_0(r,\phi)\,rdrd\phi
}{
\int r^n s_0(r,\phi)\,rdrd\phi
}

其中：

* \epsilon_n：第 n 阶空间偏心率；
* \Psi_n：第 n 阶 participant plane；
* r,\phi：横向极坐标。

至少计算：

n=2

建议保留：

n=2,3

如果后续要研究更高阶结构，可扩展到：

n=4

注意：这些几何量只用于诊断和流场响应，不用于把完整密度场替换成椭圆。

⸻

3. 横向膨胀与 freeze-out 密度

V1 的核心是对完整密度场做整体膨胀和平滑，而不是只保留协方差椭圆。

先在二阶 participant plane 坐标系中定义：

(x,y)\rightarrow (x',y')

然后做仿射膨胀：

x_f'=\lambda_x x_0'

y_f'=\lambda_y y_0'

其中：

* \lambda_x：in-plane 方向膨胀系数；
* \lambda_y：out-of-plane 方向膨胀系数。

对于非中心碰撞，通常短轴方向压力梯度更强，因此可取：

\lambda_x>\lambda_y

再加入演化平滑：

s_f(x,y)
=
G_{\sigma_{\rm evo}}
*
s_0\left(
\frac{x'}{\lambda_x},
\frac{y'}{\lambda_y}
\right)

其中：

* G_{\sigma_{\rm evo}}：Gaussian evolution smoothing kernel；
* \sigma_{\rm evo}：模拟演化过程中 hot spots 被平滑的尺度。

这一步的物理含义是：

\text{初态密度}
\rightarrow
\text{横向膨胀}
+
\text{局域结构平滑}
\rightarrow
\text{freeze-out 发射密度}

完成后可重新计算：

\epsilon_{n,f},\Psi_{n,f}

并记录：

\chi_n=\frac{\epsilon_{n,f}}{\epsilon_{n,0}}

用于检查高阶结构是否被过度抹除。

⸻

4. 纵向结构与时间结构（代码中已有固定时长，可考虑修改成有限时长(做成开关切换)）

纵向空间快度 \eta_s 使用中央平台加两侧 Gaussian 尾：

H_\eta(\eta_s)
=
\begin{cases}
1,
& |\eta_s|<\eta_0
\\
\exp\left[
-\frac{(|\eta_s|-\eta_0)^2}{2\sigma_\eta^2}
\right],
& |\eta_s|\ge \eta_0
\end{cases}

其中：

* \eta_0：中央平台半宽；
* \sigma_\eta：两侧 Gaussian 尾宽度。

freeze-out proper time 第一版可固定：

H_\tau(\tau)=\delta(\tau-\tau_f)

建议后续允许有限发射时长：

H_\tau(\tau)
\propto
\exp\left[
-\frac{(\tau-\tau_f)^2}{2\Delta\tau^2}
\right]

时空坐标由：

t=\tau\cosh\eta_s

z=\tau\sinh\eta_s

给出。

注意：

\tau_f

单位应为 fm/c。

⸻

5. 横向流场定义

5.1 基础径向流（代码中已有）

定义归一化横向半径：

\tilde r
=
\sqrt{
\frac{x'^2}{R_{x,f}^2}
+
\frac{y'^2}{R_{y,f}^2}
}

其中 R_{x,f},R_{y,f} 可由 s_f(x,y) 的二阶矩确定。

基础横向流快度：

\rho_{\rm base}(x,y)
=
\rho_0 \tilde r^m

其中：

* \rho_0：平均 radial flow 强度；
* m：径向流增长指数。

⸻

5.2 各向异性流响应

推荐使用指数形式，保证流快度调制因子正定：

\rho_T(x,y)
=
\rho_0 \tilde r^m
\exp
\left[
2\sum_{n=2}^{n_{\max}}
\kappa_n D_n\epsilon_n
\cos n(\phi_b-\Psi_n)
\right]

其中：

* \rho_T：横向流快度；
* \epsilon_n,\Psi_n：初态或末态的第 n 阶空间偏心率和平面；
* \kappa_n：第 n 阶流响应系数；
* D_n：高阶阻尼因子；
* \phi_b：流动方向角；
* n_{\max}：保留的最高流场谐波阶数。

第一版建议：

n_{\max}=2

即只保留二阶：

\rho_T
=
\rho_0 \tilde r^m
\exp
\left[
2\kappa_2\epsilon_2\cos2(\phi_b-\Psi_2)
\right]

高阶扩展版建议（考虑做成开关）：

n_{\max}=3

即：

\rho_T
=
\rho_0 \tilde r^m
\exp
\left[
2\kappa_2\epsilon_2\cos2(\phi_b-\Psi_2)
+
2\kappa_3D_3\epsilon_3\cos3(\phi_b-\Psi_3)
\right]

这里 \kappa_n\epsilon_n 是 response ansatz，代表：

\text{初态空间各向异性}
\rightarrow
\text{流场各向异性}

不是严格流体方程解，因此 \kappa_n 应作为扫描参数。

⸻

6. 流动方向（代码中已有）

使用 freeze-out contour 的法向方向，尤其在椭圆源中：

\phi_b=\arg(\hat n_{\rm surface})

其中 \hat n_{\rm surface} 是等密度面或等 \tilde r 面的外法向。

如果使用椭圆等半径面：

\tilde r^2
=
\frac{x'^2}{R_{x,f}^2}
+
\frac{y'^2}{R_{y,f}^2}

则法向方向可由：

\vec n
\propto
\left(
\frac{x'}{R_{x,f}^2},
\frac{y'}{R_{y,f}^2}
\right)

确定。

1. 局域四速度（代码中已有）

由横向流快度得到横向速度：

v_T=\tanh\rho_T

纵向流第一版取 Bjorken-like：

\eta_f=\eta_s

局域流体四速度为：

u^\mu
=
(
\cosh\rho_T\cosh\eta_f,\,
\sinh\rho_T\cos\phi_b,\,
\sinh\rho_T\sin\phi_b,\,
\cosh\rho_T\sinh\eta_f
)

需要保证：

u^\mu u_\mu=1

并设置最大横向速度限制：

v_T < v_{\max}

方案边界

V1 的物理含义应表述为：

\boxed{
\text{一个由初态 Glauber geometry 约束的参数化 freeze-out response model}
}

而不是：

\text{完整动力学演化模型}

其中：

s_0\rightarrow s_f

模拟空间膨胀和平滑；

\epsilon_n\rightarrow \rho_n

模拟初态几何到流场谐波的低维响应；

S(x,p)

负责从 freeze-out source 生成粒子的时空动量分布。
